# Go Mastery Exercises

This repository contains 50 progressive exercises to help you master Go, from basics to advanced system design.

The exercises are split into 5 parts:

- **[Part 1: Basics (Exercises 1-10)](./01_basics.md)**
  - Calculator, Log Analyzer, Interfaces, Concurrency, Context, JSON, Errors, HTTP API, Middleware, Rate Limiter.
- **[Part 2: Intermediate (Exercises 11-20)](./02_intermediate.md)**
  - SQLite CRUD, Cobra CLI, WebSocket, Pipeline, Context Timeout, Generics, Reflection, Testing, Composition, Worker Pool.
- **[Part 3: Advanced (Exercises 21-30)](./03_advanced.md)**
  - LRU Cache, Event Bus, Retry/Backoff, gRPC, Mini Framework, Functional Options, Atomic Ops, Escape Analysis, Deep Equal, WebAssembly.
- **[Part 4: Systems (Exercises 31-40)](./04_systems.md)**
  - Fuzzing, Profiling, Plugin System, Static Analysis, AST Modifier, Code Gen, TCP Chat, UDP Tracker, GraphQL, Clean Arch.
- **[Part 5: Capstone (Exercises 41-50)](./05_capstone.md)**
  - Event Sourcing, CQRS, Consistent Hash, Leader Election, Blockchain, Interpreter (Lexer/Parser/Eval), P2P, Microservices.
